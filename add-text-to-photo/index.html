Frequently asked questions

Is Add Text to Photo free to use?

Yes. Add Text to Photo is free, with no login and no watermark on the result.

Where does my photo get processed?

Processing happens locally in your browser tab. Files are not uploaded to a server for this tool.

How does Add Text to Photo actually work?

This tool applies real, deterministic image-processing (histogram auto-levels, unsharp masking, color-range/brush-guided blending, etc.) entirely in your browser. It's not a trained deep-learning model, but the transformation is genuine and reproducible — not a static demo.

What file types and sizes are supported?

JPG, PNG, WEBP and HEIC (auto-converted). Max file size 25MB, max resolution 40 megapixels. GIFs are accepted but only the first frame is edited.
