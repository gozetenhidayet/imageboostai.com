Frequently asked questions

Is Product Photo Editor free to use?

Yes. Product Photo Editor is free, with no login and no watermark on the result.

Where does my photo get processed?

Processing happens locally in your browser tab. Files are not uploaded to a server for this tool.

How does Product Photo Editor actually work?

This tool runs a real trained AI model directly in your browser (WASM/ONNX for background segmentation, or Tesseract.js for text recognition) — no photo is uploaded to a server to get the result.

What file types and sizes are supported?

JPG, PNG, WEBP and HEIC (auto-converted). Max file size 25MB, max resolution 40 megapixels. GIFs are accepted but only the first frame is edited.
